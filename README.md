# 🌌 Acerca del proyecto

**Luxury Event Rental System** es un sistema web orientado a la administración y renta de equipos, mobiliario y accesorios para eventos sociales y corporativos.

La plataforma permite gestionar inventario, reservaciones y disponibilidad de productos premium utilizados en bodas, fiestas, conferencias y eventos especiales.

El sistema fue diseñado para:

- 🎪 Gestionar eventos
- 🪑 Administrar mobiliario
- 💡 Gestionar decoración
- 🔊 Controlar equipos de sonido
- 📅 Administrar reservaciones
- 📊 Supervisar inventario
- 🔐 Gestionar accesos
- 🌐 Automatizar rentas y eventos

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaRentaMoviliarioEventos.git
```

---

## 2️⃣ Mover archivos

Copiar proyecto hacia:

```bash
xampp/htdocs/PlataformaRentaMoviliarioEventos/
```

---

## 3️⃣ Crear base de datos

Crear base:

```bash
luxury_event_rental
```

---

## 4️⃣ Importar SQL

Importar:

```bash
database/luxury_event_rental.sql
```

---

## 5️⃣ Configurar conexión

Editar:

```bash
includes/config.php
```

Agregar:

```php
define('DB_HOST','localhost');
define('DB_USER','root');
define('DB_PASS','');
define('DB_NAME','luxury_event_rental');
```

---

## 6️⃣ Ejecutar proyecto

Abrir:

```bash
http://localhost/PlataformaRentaMoviliarioEventos/
```
